# ARKITECH

AI-inspired college admissions command center built with React, Vite, Framer Motion, Zustand, and Tailwind CSS.

## Why It Matters

ARKITECH is a polished student-facing dashboard for planning college applications. It combines an onboarding flow, admissions radar, essay workspace, activity planning, interview prep, financial planning, and an assistant-style chat experience into one cohesive product prototype.

## Features

- Landing page and guided onboarding flow.
- Persistent profile and app state with Zustand.
- Dashboard shell with admissions, essays, activities, college list, financial aid, test prep, and interview modules.
- Motion-rich interface using Framer Motion.
- Reusable component structure for product sections and dashboard tools.
- Vite production build with ESLint configured.

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Framer Motion
- Zustand
- React Router
- Lucide React

## Run Locally

```bash
npm install
npm run dev
```

## Verify

```bash
npm run build
npm run lint
```

## Environment

The app can run as a frontend prototype without real AI credentials. If you wire it to an API, create a local `.env` file and keep it out of git.
