## What is ARKITECH?

ARKITECH is a polished student-facing dashboard for planning college applications. It combines an **onboarding flow**, **admissions radar**, **essay workspace**, **activity planner**, **interview prep**, **financial aid calculator**, and an **AI assistant chat** into one cohesive product prototype.

Built like a real production app — not a wireframe.

---

## Features

| Module | Description |
|--------|-------------|
| 🎯 **Admissions Radar** | Visual breakdown of your application strength |
| ✍️ **Essay Workspace** | Draft and manage college essays |
| 📋 **Activity Planner** | Organise extracurriculars and achievements |
| 🏫 **College List** | Track target, match, and safety schools |
| 💰 **Financial Aid** | Aid estimates and cost-of-attendance planning |
| 📚 **Test Prep** | SAT/ACT study planning tools |
| 🎤 **Interview Prep** | Practice and prepare for admissions interviews |
| 🤖 **AI Assistant** | Chat-based guidance throughout the process |
| 🎬 **Motion-Rich UI** | Smooth transitions and animations via Framer Motion |

## Run Locally

```bash
git clone https://github.com/by-Kaimercer/arkitech-college-assistant.git
cd arkitech-college-assistant
npm install
npm run dev
```

Open **[http://localhost:5173](http://localhost:5173)**

> Runs as a full frontend prototype — no API credentials needed out of the box.

---

## Verify

```bash
npm run build
npm run lint
```
